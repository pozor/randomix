# Contributing

Pocket Random is proprietary software. The source code is not available.

You can contribute by:

- **Reporting bugs** — [Open an issue](https://github.com/pozor/pocket-random/issues/new?template=bug_report.md)
- **Suggesting features** — [Request a feature](https://github.com/pozor/pocket-random/issues/new?template=feature_request.md)
- **Reporting translation errors** — Include the language, incorrect text, and suggested correction
