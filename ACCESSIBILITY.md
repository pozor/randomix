# Accessibility

Pocket Random is designed for **WCAG 2.1 Level AA** compliance.

---

## Features

- **VoiceOver** — Full support with semantic labels
- **Dynamic Type** — Respects iOS text size settings
- **High Contrast** — All themes meet 4.5:1 minimum ratio
- **Touch Targets** — Minimum 48×48px
- **Reduced Motion** — Animations disabled when enabled in iOS settings
- **Shake-to-Generate** — Optional feature with button alternative (WCAG 2.5.4)

---

## Supported Languages

English, Italian, French, German, Spanish, Portuguese

---

## Report an Issue

Accessibility issues are high priority.

[Report an accessibility issue](https://github.com/pozor/pocket-random/issues/new?template=bug_report.md)

Include:
- Assistive technology used
- iOS version
- Description of the issue
- Which screen/feature
