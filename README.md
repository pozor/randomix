# Pocket Random

**Random number and phrase generator for iOS**

![Version](https://img.shields.io/badge/version-1.0.1-green)

---

## Features

**Number Generator** — Random integers with custom range, history tracking, repetition control

**Phrase Generator** — Custom word lists with weighted probabilities

**Statistical Distributions** — Normal, Exponential, Poisson, Binomial, Weibull, Triangular, Uniform

**Customization** — 4 themes (Light, Dark, Modern, Christmas) · 6 languages

**Accessibility** — WCAG 2.1 AA compliant · VoiceOver support · Dynamic Type

**Privacy** — 100% offline · No data collection · No ads
